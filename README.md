## AI Doctor: A Smart Healthcare Assistant

<h3>
This repository hosts AI Doctor, an intelligent healthcare assistant that combines TensorFlow, Random Forest Classifier, and other machine learning algorithms for disease prediction and medicine recommendations.

Backend: Powered by Flask API to handle AI-based predictions and provide reliable responses.
Frontend: Designed using the MERN Stack for an intuitive and user-friendly interface.

Key Features:
1. Symptom-based disease prediction.
2. Personalized medicine recommendations.
3. Chatbot integration for seamless user interaction.

This project demonstrates the fusion of AI and modern web technologies to enhance digital healthcare experiences.
</h3>
